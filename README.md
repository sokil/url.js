# Url.js

Library to work with urls

[![Build Status](https://travis-ci.org/sokil/url.js.svg?branch=master)](https://travis-ci.org/sokil/url.js)

## Installation

### Bower
```
bower install urlib
```

## Tests

You must install `jasmine`:
```
npm install -g jasmine
```

Then run in console:
```
cd tests
jasmine
```
